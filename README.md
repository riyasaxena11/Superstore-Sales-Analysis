\# Superstore Sales Analysis Dashboard



Data Analysis project using Excel, SQL, and Power BI.



\## Tools Used

\- Excel

\- SQL

\- Power BI

\- Python







\## Project Overview



This project analyses Superstore sales data to identify sales trends, profit performance, and customer segment insights. The analysis was performed using Excel, Python, SQL, and Power BI to create an interactive dashboard.



\## Project Workflow



\- Data cleaning and exploration using Python and Excel

\- Data analysis using SQL queries

\- Business insights generation

\- Interactive dashboard creation using Power BI



\## Key Insights



\- The West region generated the highest sales.

\- The technology category contributed the highest profit.

\- Consumer segment generated the highest sales.

\- Some sub-categories showed negative profit and require further analysis.
